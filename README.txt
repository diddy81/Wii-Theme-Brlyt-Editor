Wii-Theme-Brlyt-Editor
======================

Easily edit Wii theme brlyt colours

This program is designed to be used in conjunction with the Wii Theme Team Base Pack and Thememii MOD
both can be downloaded in the program

Some functions may work with other theme bases but not all


How to use the program
======================

First thing you need to do is set your brlyt folder so the program can find the files it needs to edit
this is the brlyt folder inside your theme base pack 

Next select the menu part you want to edit from the drop down boxes then click open colour editer
